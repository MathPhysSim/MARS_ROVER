# Week 1: Introduction to Reinforcement Learning

## Exercise Set 1

This folder contains the exercises for the first week of the Introduction to Reinforcement Learning course.

### Learning Objectives

By completing these exercises, you will:
- Understand the basic components of a reinforcement learning problem
- Gain familiarity with the Gymnasium library and its API
- Learn how to create and interact with the Mars Rover environment
- Implement a simple agent that interacts with an environment

### Exercise 1: Environment Exploration

In this exercise, you will explore the Mars Rover environment and understand its dynamics:

1. Create a Mars Rover environment with different parameters
2. Run episodes with random actions and observe the outcomes
3. Analyze the transition probabilities and rewards
4. Visualize the results

### Exercise 2: Building a Simple Agent

In this exercise, you will implement a simple agent that makes decisions based on fixed rules:

1. Create an agent that always moves right
2. Create an agent that alternates between left and right
3. Create an agent that makes decisions based on its current state
4. Compare the performance of these agents

### Exercise 3: Markov Property Investigation

In this exercise, you will investigate the Markov property of the Mars Rover environment:

1. Analyze whether the environment satisfies the Markov property
2. Experiment with different configurations that might violate the Markov property
3. Discuss the implications for reinforcement learning algorithms
